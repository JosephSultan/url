## 🔧 Setup
### Clone this repository:
`git clone https://github.com/anonurl/anonurl`

### Move to repository:
`cd anonurl`

### Install all dependencies:
`npm install`

### Build:
`npm run build`

### Run (Build):
`npm run start:build`

### Run (Dev):
`npm run start`
